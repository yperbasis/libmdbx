642  *
643  * - Modern Windows versions, including Windows Vista and later, provides
644  * support for "TLS Directory" (e.g .CRT$XL[A-Z] sections in executable

646  * therefore you DON'T NEED to call mdbx_module_handler()

652  * so the MDBX_MANUAL_MODULE_HANDLER defined as 1.

656  * the MDBX_MANUAL_MODULE_HANDLER. */
